
# Alpha Space

![](https://github.com/sugarvoid/alpha-space/blob/master/docs/gameplay.gif)

## Goal
Use the random letters to make words

## Controls

|Button|Action|
| ----------| ---------|
|Left Click | Select Letter|
|Right Click | Store Letter|
